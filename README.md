# 다카마쓰 트립 2026

2026년 연세대 산업공학과 시스템 인텔리전스 랩의 다카마쓰 여행을 위한 웹 앱입니다.

## 기능

- 🗓️ 여행 일정 관리
- 📋 체크리스트
- 🗣️ 일본어 회화 학습
- 🎵 여행 음악 플레이어
- 🌤️ 다카마쓰 날씨 정보
- ✈️ 항공 및 숙박 정보

## 기술 스택

- React + TypeScript
- Vite
- Tailwind CSS
- Motion (Framer Motion)
- shadcn/ui

## 로컬 개발

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드 미리보기
npm run preview
```

## 배포

이 프로젝트는 Vercel, Netlify, GitHub Pages 등 다양한 플랫폼에 배포할 수 있습니다.
